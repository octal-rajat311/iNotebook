# iNotebook - Secured Notes on Cloud
## **Tech Stack** : 
* HTML
* CSS
* JavaScript
* MongoDb
* Express Js
* Node Js
* React Js
* Bootstrap
* Git / GitHub
___
## **Description** : 
* Stored on cloud
* Secured
* Notes categorised on basis of tags
* Notes can be edited or deleted
* Completely free
